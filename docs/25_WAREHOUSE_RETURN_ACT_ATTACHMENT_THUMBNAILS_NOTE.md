# 25 Warehouse — Return Act Attachment Thumbnails Note

## 10.177. Attachment thumbnails in the return act

Decision: problem-row photos and files should be shown as small thumbnails in the return act.

Rules:
- show small thumbnails for photos where possible;
- show compact file cards or file icons for non-image files;
- thumbnails are connected to the related problem row;
- if there are many attachments, keep the layout compact;
- if no attachments exist, do not show an empty attachment area.

Purpose:
- make problem evidence visible directly in the return act;
- avoid a document that only has unreadable file links;
- keep the act readable and compact.

MVP approach:
- render small previews for image attachments;
- render compact file placeholders for non-image attachments;
- keep full files available through stored attachment links in the system.
