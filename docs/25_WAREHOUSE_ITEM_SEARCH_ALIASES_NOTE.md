# 25.x. Синонимы и альтернативные названия для поиска

Этот документ дополняет `docs/25_WAREHOUSE_AND_INVENTORY.md` и фиксирует решение 10.45.

## Решение

Для складских позиций нужно заложить синонимы и альтернативные названия как необязательное поле для поиска.

Синонимы не должны менять основное название позиции.

Основное название используется в:

- смете;
- сборочном листе;
- акте выдачи;
- акте возврата;
- карточке склада;
- внутренних документах.

Синонимы используются только для поиска, подсказок и защиты от дублей.

## Зачем это нужно

Синонимы помогают найти одну и ту же позицию разными словами, сокращениями или разговорными названиями.

Это снижает риск, что пользователи заведут одну и ту же позицию несколько раз под разными названиями.

## Правило MVP

Поле синонимов:

- необязательное;
- не влияет на расчёты;
- не влияет на документы;
- не влияет на складские остатки;
- не заменяет основное название позиции;
- используется только для поиска и подсказок.

Если поле не заполнено, позиция работает как обычно.
