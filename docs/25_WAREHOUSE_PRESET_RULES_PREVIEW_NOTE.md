# 25 Warehouse — Preset Rules Preview Note

## 10.114. Preview preset rules before save

Decision: before saving a selected warehouse rules preset, the admin should see a preview of resulting process rules.

Rules:
- available to admin/director;
- show affected warehouse processes;
- show current value and preset value for each affected process;
- allow confirm or cancel;
- unchanged processes may be hidden or grouped separately.

MVP:
- simple table or list is enough;
- no advanced visual diff is required;
- after saving, process rules remain editable manually.
