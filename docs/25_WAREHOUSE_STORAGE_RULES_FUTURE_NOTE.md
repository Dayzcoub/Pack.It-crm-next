# 25.x. Инструкция по укладке и правила хранения

Этот документ дополняет `docs/25_WAREHOUSE_STORAGE_PLACE_PHOTOS_NOTE.md` и фиксирует решение 10.58.

## Решение

Инструкцию по укладке или правило хранения стоит заложить как будущую опциональную функцию.

Для MVP это не обязательный элемент, но в следующих итерациях такая возможность может быть полезной, особенно вместе с фото зон, стеллажей и полок.

## Возможная логика

К месту хранения можно будет добавить текстовое поле с правилом хранения.

Примеры:

- как правильно складывать оборудование;
- что можно хранить на этой полке;
- что нельзя ставить в эту зону;
- как должен выглядеть порядок после возврата;
- особенности хранения хрупких или тяжёлых позиций.

## Связь с фото места хранения

Фото показывает, как место выглядит визуально.

Текстовая инструкция объясняет правило или нюанс, который не всегда очевиден по фото.

Вместе они могут работать как простая инструкция для новых сотрудников и профильных специалистов.

## MVP-статус

На старте достаточно:

- фото места хранения;
- комментарий к месту хранения;
- структурированное место хранения.

Отдельное поле `инструкция по укладке / правило хранения` можно добавить позже без изменения основной складской логики.
