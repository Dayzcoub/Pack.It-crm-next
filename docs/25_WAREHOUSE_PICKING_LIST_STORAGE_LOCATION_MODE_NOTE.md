# 25 Warehouse — Picking List Storage Location Mode Note

## 10.138. Место хранения в сборочном листе

Решение: сборочный лист по умолчанию должен быть упрощённым, без подробного места хранения. Расширенный вариант с местами хранения можно включить позже в админ-настройках.

Правило:
- базовый сборочный лист показывает позиции и количество;
- подробное место хранения не выводится по умолчанию;
- когда компания наведёт порядок на складе и заполнит места хранения, администратор сможет включить расширенный режим;
- расширенный режим может показывать склад, зону, стеллаж, полку или ячейку.

Причина:
- на старте у многих компаний места хранения будут заполнены не полностью или неаккуратно;
- если сразу выводить неправильные места, сборочный лист будет мешать работе;
- упрощённый документ надёжнее для MVP;
- расширенный режим полезен позже, когда складовая структура реально поддерживается.

MVP-подход:
- default picking list mode = simple;
- simple mode: item, quantity, category, comments, check status;
- extended location mode is optional and controlled by admin setting;
- no prices in any picking list mode.
