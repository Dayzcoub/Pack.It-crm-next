# 25.x. Ответственные за складские операции

Этот документ дополняет `docs/25_WAREHOUSE_AND_INVENTORY.md` и фиксирует решение 10.31.

## Решение

Для складских операций нужно заложить возможность указывать отдельных ответственных за действия склада, но эта логика должна быть настраиваемой.

Для больших складов и компаний со складскими работниками полезны отдельные поля:

- кто собрал;
- кто выдал;
- кто принял возврат.

Для маленькой компании, где одни и те же люди делают почти всё сами, достаточно общего автора операции.

Поэтому отдельные поля ответственных должны включаться глобально в админке программы.

Если настройка выключена, система показывает упрощённый режим: автор операции и общий комментарий.

Если настройка включена, складские документы и операции могут показывать отдельных ответственных по этапам.

Это позволяет не перегружать маленькие команды, но оставить нормальный режим учёта для больших складов.
