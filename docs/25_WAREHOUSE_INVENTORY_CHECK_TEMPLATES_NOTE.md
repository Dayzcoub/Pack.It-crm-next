# 25.x. Шаблоны инвентаризационных проверок

Этот документ дополняет `docs/25_WAREHOUSE_INVENTORY_AUDIT_NOTE.md` и фиксирует решение 10.93.

## Решение

Для инвентаризации нужны шаблоны проверок.

Шаблон позволяет не выбирать каждый раз вручную область проверки, категории и фильтры.

## Примеры шаблонов

Шаблоны могут описывать повторяемые проверки:

- инвентаризация зоны звук;
- инвентаризация кабелей;
- инвентаризация LED перед сезоном;
- инвентаризация ферм после большого проекта;
- проверка конкретного склада;
- проверка стеллажа или группы стеллажей;
- проверка позиций с проблемами.

## Что хранит шаблон

Шаблон может хранить:

- название шаблона;
- складскую локацию;
- зону;
- стеллаж / полку, если нужно;
- категории и подкатегории;
- ответственного по умолчанию;
- комментарий;
- настройки видимости и доступа.

## Использование

Пользователь выбирает шаблон и запускает инвентаризацию.

Система автоматически подставляет область и фильтры проверки.

Перед запуском пользователь может уточнить параметры, если это разрешено.

## MVP

Для MVP достаточно:

1. создать шаблон проверки;
2. выбрать область и фильтры;
3. сохранить шаблон;
4. запускать инвентаризацию из шаблона;
5. при необходимости редактировать шаблон пользователем с правами.
