# 25.x. Комментарий к месту передачи

Этот документ дополняет `docs/25_WAREHOUSE_AND_INVENTORY.md` и фиксирует решение 10.30.

## Решение

Для складских операций не вводим отдельное обязательное поле места передачи.

На старте достаточно обычного комментария к операции.

В этом комментарии пользователь может указать, где фактически передали или приняли имущество.

Такой подход не усложняет складскую операцию лишними обязательными полями, но оставляет возможность зафиксировать важную информацию.

Если позже понадобится более строгая логистика, отдельное структурированное поле места передачи можно добавить в будущих версиях.
