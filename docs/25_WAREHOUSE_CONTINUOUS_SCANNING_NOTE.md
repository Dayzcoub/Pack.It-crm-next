# 25.x. Непрерывное сканирование

Этот документ дополняет `docs/25_WAREHOUSE_MOBILE_SCANNER_MODE_NOTE.md` и фиксирует решение 10.74.

## Решение

В мобильном сканере нужен режим непрерывного сканирования как опция.

Режим не должен быть включён всегда, но должен быть доступен для операций, где нужно быстро обработать много объектов подряд.

## Где использовать

Режим полезен для:

- сборки проекта;
- выдачи оборудования;
- возврата;
- инвентаризации;
- проверки кейсов и контейнеров.

## Логика

Пользователь включает непрерывное сканирование в текущей складской операции.

После успешного скана система выполняет действие текущего режима и оставляет экран сканера открытым.

Если режим выключен, после скана показывается обычное быстрое меню действий.

## Контроль

Если объект не подходит для текущей операции, уже отмечен, не найден или у пользователя нет прав, система показывает предупреждение и не выполняет действие автоматически.

## MVP

Для MVP достаточно:

1. переключатель `Непрерывное сканирование`;
2. выбор текущего режима действия;
3. подтверждение успешного скана;
4. список последних сканов;
5. предупреждение при ошибке.
