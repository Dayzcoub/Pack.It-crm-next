# 25 Warehouse — Handout Act Cases View Mode Note

## 10.159. Cases and containers in the handout act

Decision: cases and containers in the handout act should support both display modes.

Rules:
- cases and containers can be shown as a separate block;
- cases and containers can also be included in the common item table;
- the user may choose the mode during document generation;
- the company may also define a default mode in warehouse document settings.

Purpose:
- a separate block is clearer when cases are important for logistics;
- a common table is simpler for small projects;
- different teams can use the format that fits their warehouse process.

MVP approach:
- support separate cases / containers block;
- support common table mode;
- keep the handout act internal;
- do not include prices by default.
