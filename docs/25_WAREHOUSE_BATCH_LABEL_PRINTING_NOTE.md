# 25.x. Массовая печать наклеек

Этот документ дополняет `docs/25_WAREHOUSE_LABEL_PRINT_SETTINGS_NOTE.md` и фиксирует решение 10.69.

## Решение

В системе нужна массовая печать QR / штрихкод-наклеек.

Пользователь должен иметь возможность выбрать группу объектов и сформировать лист наклеек для печати.

## Варианты выбора

Массовую печать можно запускать по:

- категории;
- подкатегории;
- складской локации;
- зоне;
- стеллажу;
- полке / месту хранения;
- выбранному списку позиций;
- выбранному списку экземпляров;
- выбранному списку кейсов / контейнеров.

## Результат

Система формирует файл для печати, например PDF-лист с наклейками.

Наклейки используют выбранный шаблон для соответствующего типа объекта.

## Настройки

Администратор может настроить:

- шаблон наклейки;
- какие поля печатать;
- размер наклейки;
- количество наклеек на листе;
- тип кода: QR, штрихкод или оба варианта.

## MVP

Для MVP достаточно:

1. выбрать группу объектов;
2. выбрать шаблон наклейки;
3. сформировать PDF для печати;
4. напечатать или скачать файл.

Сложный дизайнер печатных макетов на старте не обязателен.
