# 25.x. Непрерывное сканирование при инвентаризации

Этот документ дополняет `docs/25_WAREHOUSE_CONTINUOUS_SCANNING_NOTE.md` и фиксирует решение 10.82.

## Решение

При инвентаризации через мобильный сканер нужно разрешить массовое подтверждение `найдено на месте` через непрерывное сканирование.

Это отдельный полезный сценарий, отличающийся от массовой смены места хранения.

## Логика

Пользователь открывает инвентаризацию выбранной области, например зоны, стеллажа или полки.

Затем включает режим непрерывного сканирования.

Каждый успешно отсканированный объект автоматически отмечается как найденный в рамках текущей инвентаризации.

## Ограничения

Автоматическая отметка должна работать только внутри активной инвентаризации.

Если объект не относится к текущей области проверки, уже отмечен, не найден в базе или имеет конфликт статуса, система должна показать предупреждение.

## Что сохранять

Для каждой отметки нужно сохранить:

- объект;
- статус `найдено`;
- кто отсканировал;
- время скана;
- инвентаризация, к которой относится отметка.

## MVP

Для MVP достаточно:

1. открыть инвентаризацию области;
2. включить непрерывное сканирование;
3. сканировать объекты подряд;
4. автоматически отмечать их как найденные;
5. показывать предупреждение при ошибке или несоответствии.
