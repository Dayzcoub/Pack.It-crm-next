# 25 Warehouse — Documents MVP Set Note

## 10.134. Базовый набор складских документов для MVP

Решение: для MVP складского блока нужен минимальный набор складских документов.

Базовый набор:
- сборочный лист;
- акт выдачи;
- акт возврата;
- отчёт инвентаризации;
- внутренний отчёт по проблемам и потерям.

Правило:
- складские документы должны покрывать основные рабочие сценарии: собрать, выдать, вернуть, проверить, разобрать проблемы;
- не нужно сразу делать большой документооборот;
- каждый документ должен быть связан с соответствующей складской операцией или проектом;
- документы могут быть доступны для PDF/печати по правам.

Архитектурный смысл:
- сборочный лист помогает подготовить комплект под проект;
- акт выдачи фиксирует передачу оборудования;
- акт возврата фиксирует факт возврата и проблемные позиции;
- отчёт инвентаризации фиксирует результат проверки склада;
- внутренний отчёт по проблемам/потерям помогает разбирать спорные и критичные случаи.

MVP-подход:
- сделать эти пять типов документов основой складского документооборота;
- дополнительные документы добавлять позже по реальной необходимости;
- держать документы внутренними, если они не предназначены для клиента.
