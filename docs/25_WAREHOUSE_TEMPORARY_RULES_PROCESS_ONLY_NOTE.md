# 25 Warehouse — Temporary Rules Process Only Note

## 10.128. Область действия временных правил склада

Решение: временные правила склада не нужно детализировать по категориям оборудования.

Правило:
- временные правила задаются на уровне складского процесса;
- временные правила не задаются отдельно для категорий оборудования;
- не делать сценарий вроде `строгий возврат только для LED` на MVP;
- если процесс временно ужесточается, он применяется к процессу целиком.

Причина:
- детализация по категориям оборудования перегружает настройки;
- временные правила должны оставаться простым административным инструментом;
- для MVP достаточно уровня процесса: выдача, возврат, инвентаризация, списания, проблемные строки и т.д.

MVP-подход:
- temporary rules scope = process only;
- no category-level temporary rules;
- no equipment-type exceptions for temporary rules at the start.
