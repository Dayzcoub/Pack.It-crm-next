# 25.x. Быстрое меню после сканирования QR или штрихкода

Этот документ дополняет `docs/25_WAREHOUSE_QR_CODE_PAYLOAD_NOTE.md` и фиксирует решение 10.72.

## Решение

После сканирования QR-кода или штрихкода пользователь должен видеть быстрое меню действий.

Сканирование не должно ограничиваться только открытием карточки объекта.

## Логика

Система определяет объект по QR / штрихкоду и показывает действия, доступные пользователю по роли и контексту.

Примеры действий:

- открыть карточку;
- добавить в сборку;
- выдать;
- принять возврат;
- отметить на инвентаризации;
- открыть документы и фото;
- зафиксировать проблему, если есть права.

## Контекст

Набор действий может зависеть от текущего режима работы:

- обычный поиск;
- сборочный лист;
- выдача;
- возврат;
- инвентаризация;
- ремонт / сервис;
- карточка места хранения.

## Права доступа

Пользователь видит только те действия, которые ему разрешены.

Например, профильный сотрудник может отметить позицию на инвентаризации, но не может подтвердить списание.

## MVP

Для MVP достаточно после скана показывать меню:

- открыть карточку;
- добавить / отметить в текущей складской операции, если она открыта;
- отметить на инвентаризации, если открыт режим инвентаризации;
- зафиксировать проблему, если действие разрешено.

Если нет активного складского режима, скан может по умолчанию открывать карточку объекта.
