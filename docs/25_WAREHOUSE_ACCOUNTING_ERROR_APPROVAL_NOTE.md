# 25 Warehouse — Accounting Error Approval Note

## 10.260. Accounting error correction requires approval

Decision: shortage closure by accounting error correction requires admin/director approval.

Terminology:
- use clearer closure result name: accounting error corrected;
- avoid unclear wording like manual correction where possible.

Rules:
- warehouse keeper or assigned responsible person can propose accounting error correction;
- correction proposal must include required reason/comment;
- proposal records who suggested the correction;
- admin/director must approve before the shortage is finally closed;
- if approved, shortage is closed with accounting error corrected result;
- if rejected, shortage remains open with rejection reason;
- all actions are logged.

Purpose:
- prevent uncontrolled stock corrections;
- keep responsibility clear;
- preserve explanation of why the accounting quantity was corrected.

MVP approach:
- correction reason is required;
- create approval request for admin/director;
- final closure happens only after approval.
