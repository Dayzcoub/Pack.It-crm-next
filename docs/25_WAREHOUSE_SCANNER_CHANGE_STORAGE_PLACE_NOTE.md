# 25.x. Изменение места хранения из мобильного сканера

Этот документ дополняет `docs/25_WAREHOUSE_SCANNER_FIND_STORAGE_PLACE_NOTE.md` и фиксирует решение 10.78.

## Решение

Из мобильного сканера нужно разрешить быстро изменить место хранения позиции, экземпляра, кейса или контейнера.

Это удобно для синхронизации фактической расстановки на стеллажах с данными в системе.

## Логика

Пользователь сканирует объект и видит текущее место хранения.

Если объект находится не там, пользователь может выбрать действие `Изменить место хранения`.

После этого он выбирает новое место:

- складскую локацию;
- зону;
- стеллаж;
- полку / уровень;
- ячейку, если используется.

## Права доступа

Изменение места хранения должно быть доступно только пользователям с разрешением на складские операции или инвентаризацию в соответствующей области.

Обычный пользователь без складских прав не должен менять место хранения.

## История

Изменение места хранения должно сохраняться в истории объекта:

- кто изменил;
- когда изменил;
- старое место;
- новое место;
- комментарий, если указан.

## MVP

Для MVP достаточно:

1. действие `Изменить место хранения` после скана;
2. выбор нового места хранения из структуры склада;
3. сохранение изменения в карточке объекта;
4. запись в историю объекта.

Это не полноценный журнал внутренних перемещений, а быстрый способ актуализировать место хранения при работе со складом.
