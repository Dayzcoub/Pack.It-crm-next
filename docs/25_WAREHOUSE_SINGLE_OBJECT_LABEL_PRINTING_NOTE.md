# 25.x. Печать наклейки из карточки объекта

Этот документ дополняет `docs/25_WAREHOUSE_BATCH_LABEL_PRINTING_NOTE.md` и фиксирует решение 10.70.

## Решение

Печать наклейки должна быть доступна прямо из карточки объекта.

Это нужно для оперативности: пользователь открыл конкретную позицию, экземпляр, кейс или место хранения и сразу может сформировать наклейку.

## Где нужна кнопка печати

Кнопку `Печать наклейки` стоит предусмотреть в карточках:

- общей складской позиции;
- конкретного экземпляра;
- кейса / контейнера;
- стеллажа / полки / места хранения.

## Логика

Система выбирает подходящий шаблон наклейки по типу объекта.

Пользователь может сразу сформировать наклейку для текущего объекта.

Если администратор настроил несколько шаблонов для такого типа объекта, пользователь может выбрать нужный шаблон.

## MVP

Для MVP достаточно:

1. кнопка `Печать наклейки` в карточке объекта;
2. автоматический выбор базового шаблона;
3. формирование PDF или печатного макета для одной наклейки.

Массовая печать остаётся отдельной функцией для групп объектов.
