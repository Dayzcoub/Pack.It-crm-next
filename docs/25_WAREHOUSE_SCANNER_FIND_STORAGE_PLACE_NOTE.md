# 25.x. Режим поиска места хранения в мобильном сканере

Этот документ дополняет `docs/25_WAREHOUSE_MOBILE_SCANNER_MODE_NOTE.md` и фиксирует решение 10.77.

## Решение

В мобильном сканере нужен режим `найти место хранения`.

После сканирования позиции, экземпляра, кейса или контейнера система должна быстро показать, где объект должен храниться.

## Что показывать

После скана система может показать:

- складскую локацию;
- зону;
- стеллаж;
- полку / уровень;
- ячейку, если указана;
- фото места хранения, если оно есть;
- кнопку `Показать на плане`, если объект привязан к визуальному 2D-плану склада.

## Зачем это нужно

Режим помогает:

- быстро найти нужную позицию на складе;
- вернуть оборудование на правильное место;
- помочь новым сотрудникам ориентироваться в складе;
- уменьшить ошибки при возврате и инвентаризации.

## MVP

Для MVP достаточно:

1. сканировать объект;
2. показать текстовый путь места хранения;
3. показать фото места, если оно есть;
4. дать кнопку перехода к плану склада, если план настроен.

Если место хранения не указано, система должна показать предупреждение и предложить открыть карточку объекта.
