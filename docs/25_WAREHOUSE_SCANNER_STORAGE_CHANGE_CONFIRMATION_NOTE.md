# 25.x. Подтверждение смены места хранения из сканера

Этот документ дополняет `docs/25_WAREHOUSE_SCANNER_STORAGE_PLACE_AUTO_HISTORY_NOTE.md` и фиксирует решение 10.80.

## Решение

При смене места хранения из мобильного сканера нужно показывать подтверждение перед сохранением.

Пользователь должен видеть, какой объект перемещается, откуда и куда.

## Что показывать в подтверждении

Перед сохранением система показывает:

- название объекта;
- внутренний код;
- номер экземпляра, если применимо;
- старое место хранения;
- новое место хранения;
- кнопку подтверждения;
- кнопку отмены.

## Почему так

Смена места хранения влияет на фактическую организацию склада.

Подтверждение снижает риск случайно перезаписать место хранения не тому объекту или выбрать не ту полку.

## Комментарий

Комментарий при подтверждении не обязателен.

Система автоматически сохраняет старое и новое место, пользователя и дату изменения.

## MVP

Для MVP достаточно простого экрана подтверждения:

```text
Переместить:
SND-000245 / Shure QLXD #003

из:
Склад 1 → Зона звук → Стеллаж A → Полка 2

в:
Склад 1 → Зона звук → Стеллаж B → Полка 1

[Подтвердить] [Отмена]
```
