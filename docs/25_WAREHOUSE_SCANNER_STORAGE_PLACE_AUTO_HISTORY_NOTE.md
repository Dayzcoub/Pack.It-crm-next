# 25.x. Автоматическая история смены места хранения из сканера

Этот документ дополняет `docs/25_WAREHOUSE_SCANNER_CHANGE_STORAGE_PLACE_NOTE.md` и фиксирует решение 10.79.

## Решение

При изменении места хранения из мобильного сканера комментарий не должен быть обязательным.

Система должна автоматически сохранить ключевые данные изменения.

## Что фиксировать автоматически

При смене места хранения нужно записать:

- кто изменил;
- когда изменил;
- старое место хранения;
- новое место хранения;
- объект, для которого изменили место.

## Комментарий

Комментарий можно оставить опциональным.

Если пользователю нужно пояснить причину изменения, он может добавить комментарий вручную.

Если комментарий не указан, операция всё равно сохраняется.

## Почему так

Главная задача этого действия — быстро синхронизировать фактическую расстановку на стеллажах с данными в системе.

Обязательный комментарий замедлит работу и будет лишним в большинстве случаев.

## MVP

Для MVP достаточно автоматической записи старого и нового места хранения в истории объекта.
