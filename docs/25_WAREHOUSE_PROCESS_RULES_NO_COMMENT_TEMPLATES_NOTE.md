# 25 Warehouse — Process Rules No Comment Templates Note

## 10.122. Шаблонные комментарии для изменения правил склада

Решение: шаблонные комментарии при ручном изменении правил склада не нужны.

Правило:
- администратор пишет комментарий вручную;
- быстрые заготовки причин не добавляем;
- выпадающий список шаблонов не нужен;
- комментарий должен быть коротким пояснением конкретной ситуации.

Причина:
- шаблоны могут добавить лишнюю сложность в настройки;
- живой комментарий лучше объясняет реальную причину изменения;
- для MVP достаточно обычного обязательного текстового поля.

MVP-подход:
- поле комментария обязательно;
- без справочника шаблонов;
- без быстрых кнопок причин;
- текст сохраняется в истории ручного изменения правил.
