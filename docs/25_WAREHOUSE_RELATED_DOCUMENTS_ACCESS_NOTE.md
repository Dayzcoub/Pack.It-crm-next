# 25.x. Доступ к связанным документам складской позиции

Этот документ дополняет `docs/25_WAREHOUSE_RELATED_DOCUMENTS_NOTE.md` и фиксирует решение 10.61.

## Решение

Для связанных документов и материалов складской позиции нужны уровни доступа.

Разные типы материалов имеют разную чувствительность, поэтому доступ к ним должен зависеть от роли пользователя и типа документа.

## Примеры доступа

Мануал или инструкция может быть доступна техникам и складским сотрудникам.

Чек, счёт, договор покупки и документы с закупочной стоимостью должны быть доступны только администратору, директору или пользователю с финансовым доступом.

Фото повреждений или проблем может быть доступно складу, менеджеру проекта, администратору и директору.

Гарантийные и сервисные документы могут быть доступны складу, техническим ролям, администратору и директору.

## Логика

Для каждого материала можно хранить:

- тип материала;
- уровень видимости;
- роли, которым разрешён доступ;
- комментарий;
- кто добавил;
- дату добавления.

## MVP

Для MVP достаточно простых уровней доступа:

- все пользователи с доступом к карточке;
- технические роли и склад;
- склад и менеджеры;
- только админ / директор;
- только пользователи с финансовым доступом.

Если уровень доступа не указан, можно использовать безопасное значение по умолчанию на основе типа материала.
