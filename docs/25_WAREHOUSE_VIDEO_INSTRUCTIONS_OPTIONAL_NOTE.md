# 25.x. Видеоинструкции для складских позиций

Этот документ дополняет складскую архитектуру PACK.IT CRM Next и фиксирует решение 10.59.

## Решение

Видеоинструкция не нужна как обязательная часть карточки складской позиции.

При этом возможность добавить видеоинструкцию, ссылку на видео или файл нужно предусмотреть в архитектуре.

## Логика

Видео может быть полезно для отдельных сложных позиций, но не должно быть обязательным полем.

В будущем к карточке позиции, экземпляра, комплекта или места хранения можно будет добавить:

- ссылку на видео;
- файл видео;
- комментарий к видео;
- тип инструкции: сборка, упаковка, подключение, обслуживание, другое.

## MVP

Для MVP видеоинструкции не являются обязательной функцией.

Достаточно заложить возможность будущего расширения рядом с файлами, фото, мануалами и комментариями.

Если видео не добавлено, карточка позиции работает как обычно.
