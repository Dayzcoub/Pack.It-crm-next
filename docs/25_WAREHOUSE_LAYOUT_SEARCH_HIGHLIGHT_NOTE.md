# 25.x. Поиск позиции на плане склада

Этот документ дополняет `docs/25_WAREHOUSE_LAYOUT_CONSTRUCTOR_NOTE.md` и фиксирует решение 10.55.

## Решение

В визуальном плане склада нужен быстрый поиск позиции с подсветкой места хранения.

Пользователь вводит название позиции или поисковое слово, а система показывает найденные позиции и подсвечивает их место на 2D-плане склада.

## Что подсвечивать

Если позиция привязана к плану, система может подсветить:

- складскую локацию;
- зону;
- стеллаж;
- полку / уровень;
- ячейку, если она указана.

## Связь с поиском

Поиск должен учитывать основное название позиции и необязательные синонимы.

Подсветка на плане не меняет название позиции и не влияет на документы, расчёты или остатки.

## MVP

Для MVP достаточно:

1. поле поиска в визуальном плане склада;
2. список найденных позиций;
3. подсветка зоны или стеллажа на 2D-плане;
4. отображение точного места хранения в карточке результата.

Если у позиции нет привязки к плану, система показывает обычное текстовое место хранения из карточки позиции.
