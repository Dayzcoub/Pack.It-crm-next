# 25.x. Опциональные машинные метки склада

Этот документ дополняет `docs/25_WAREHOUSE_AND_INVENTORY.md` и фиксирует решение 10.28.

## Решение

В архитектуру склада нужно заложить опциональный режим работы с машинными метками: QR-кодами и штрихкодами.

Ручной режим остаётся базовым. Склад должен уметь работать без обязательной маркировки каждой позиции.

Если у экземпляра, кейса, рэка, ящика или другой складской единицы есть машинная метка, её можно использовать при:

- сборке;
- выдаче;
- возврате;
- поиске карточки позиции;
- проверке содержимого контейнера;
- фиксации проблемной строки.

Метки должны ускорять работу и снижать риск путаницы, но не должны быть обязательным условием работы склада в MVP.

Если позиция требует поэкземплярного учёта, метка может использоваться как быстрый способ выбрать конкретный экземпляр.

Если позиция учитывается количеством, работа по меткам не обязательна.
