# 25.x. Подтверждение результата сканирования

Этот документ дополняет `docs/25_WAREHOUSE_MOBILE_SCANNER_MODE_NOTE.md` и фиксирует решение 10.75.

## Решение

В мобильном сканере нужны визуальные, звуковые и вибро-подтверждения результата сканирования.

Это помогает быстро понимать результат складской операции без внимательного чтения экрана.

## Варианты подтверждения

Система может использовать:

- зелёную визуальную отметку для успешного действия;
- красную визуальную отметку для ошибки;
- короткий звук успешного скана;
- звук ошибки;
- короткую вибрацию при успешном действии;
- более заметную вибрацию при проблеме.

## Настройки

Звук и вибрация должны быть настраиваемыми.

Пользователь или компания могут отключить звук, вибрацию или оставить только визуальное подтверждение.

## MVP

Для MVP достаточно:

1. визуальная отметка успеха;
2. визуальная отметка ошибки;
3. короткое подтверждение успешного скана;
4. отдельное предупреждение при ошибке;
5. возможность отключить звук или вибрацию, если устройство это поддерживает.
