# PACK.IT CRM Next

**PACK.IT CRM Next** — новое приложение для проектно-сметной работы, технической комплектации, склада, документов и опционального 3D-конструктора.

Проект разрабатывается с нуля. Старое приложение используется только как донор проверенной логики, формул, бизнес-сценариев и справочников. Новый продукт, документация, интерфейс и архитектура используют только бренд **PACK.IT / ПАК.ИТ**.

## Главный принцип

PACK.IT — это не 3D-конструктор как главный продукт.

PACK.IT — это **проектно-сметная CRM**, где главная сущность — **Проект / Смета**.

3D-конструктор является опциональным модулем проекта и подключается только тогда, когда он действительно нужен: сцена, фермы, LED, техническая схема, расстановка оборудования или визуализация для клиента/команды.

## Базовая формула продукта

```text
Project / Quote
→ Sections
→ Optional 3D Scene
→ BOM
→ Inventory / Reservations
→ Documents
→ Status / History / CRM
```
